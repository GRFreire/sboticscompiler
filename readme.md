## :information_source: How to use
```bash
  # Install the package globally
  $ npm -g install sboticscompiler
  # or
  $ yarn global add sboticscompiler

  # Create a new project
  $ sboticscompiler init

  # Compile your code
  $ sboticscompiler compile
```
